#  Bank Management System
A Java-based application for managing banking operations like Deposit, Withdraw, and Pin Change.

###  Tech Stack
* **Language:** Java (JDK 8+)
* **IDE:** NetBeans
* **Database:** MySQL
* **GUI:** Swing & AWT

###  Key Features
* User Login & Signup
* Balance Enquiry
* Cash Withdrawal & Deposit
* Mini Statement Generation
